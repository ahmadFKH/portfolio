CryptoCurrency World


Description

CryptoCurrency World is a learning platform where the user can buy/sell crypto currencies with real prices but with fake money. This project was made by me and two other colleagues in the final Hackathon within the framework of the coding bootcamp.

You can think about it as a practice before getting into the real world of crypto currencies trading.

My part was basically doing the whole Back-End side of the app, including:

  • Fetching data from an external CryptoCurrency API, for that I've created seperate server that updates the data every couple of seconds to make sure that the information in the app is up to date
  • Creating and updating the DB, creating schemas and models
  • Building the routes to handle different requests from client-side
  • Handling the Token based Authentication (still not completely finished)
Right now there is a problem with charts we are trying to fix it !!

Features

  • When the user creates new account, he'll get a fake 5000$ as a start
  • In the main page, he can see a table that includes the leading crypto currencies with real information about each one of them
  • The moment he chooses a certain currency to buy, he'll be directed to the coin page, that includes informatin about it and a calculator to help him decide the amount he wants to buy
  • The user can also visit his portfolio to check out what he has bought/sold, and porfolios of other users to see how they are performing and try to learn from them. Again, the point is to learn

Technologies


Front end:

HTML, CSS, Angular 6, Bootstrap

Back end:

Node.js, Express, Passport (Token based authentication), MongoDB, mongoose

Tools:

Deployed with Heroku, Postman
