Typing Hero


Description

Typing Hero is a platform where the user can choose a lesson (out of 5) and examine his speed in typing.

Features

  • Every time the user chooses a lesson, the app will set a timer for him besides a counter for his mistakes
  • The letter typed will be highlighted with a green color, and if the user got it wrong, the right letter will be highlighted with yellow
  • After the user is finished with the lesson, he can click on "submit" button, and he will see his current and previous results

Technologies

Front end:

HTML, CSS, JS, jQuery, Handlebars, Font-awesome, bootstrap

Back end:

Node.js, Express, MongoDB, mongoose

Tools:

Deployed with Heroku
