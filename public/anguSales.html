CRM Platform


Description

CRM-platform is a CRM (Customer Relationship Management) system that allows users to view, add and edit customers and companies. This is a very common piece of software used by most companies in the world. The project was built completely (Front-End) by Angular 5 (including Angular Materials).

Features

  • In the main page the user will find a list of customers that includes some detail about them, where he can edit or remove custmer's details
  • Clicking on the customer's name will redirect you to the customer's screen, where he can add/remove comments about the customer
  • In the navigation bar the user can reach out the companies screen, which includes details about companies. Besides, the user can add new companies

Technologies

Front end:

HTML, CSS, Angular 5 (including Angular Materials)

Back end:

Node.js, Express, mySQL, Sequelize

Tools:

Deployed with Heroku (using ClearDB), Postman, Workbench
